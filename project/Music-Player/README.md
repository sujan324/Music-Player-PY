# Music-Player
Your very own Music player developed using python3+pygame+tkinter+mutagen.

# Features 
```
1. select floder where all songs are.
2. Next song
3. previous song
4. pause any song
5. looping
```

# Requirements
```
python3
pygame
mutagen
```

# Run
```
python3 musicpy.py
```
